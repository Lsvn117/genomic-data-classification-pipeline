# Genomic Data Classification Pipeline

A structured data analytics pipeline that processes DNA mutation datasets and predicts mutation impact using feature engineering and classification models.

## Overview
The system converts raw DNA sequence data into machine-readable features, applies dimensionality reduction, and performs classification. A web interface enables users to upload datasets and receive predictions instantly.

The project demonstrates real-world data workflow design including ingestion, preprocessing, modeling, and reporting.

## Features
- Structured dataset creation from raw sequences
- Feature extraction and dimensionality reduction
- Automated classification pipeline
- Real-time prediction interface
- Reusable dataset generation

## Tech Stack
Python • NumPy • Pandas • Scikit-learn • Flask

## Pipeline
Input Data → Cleaning → Feature Extraction → Model → Prediction

## Concepts Demonstrated
- Data preprocessing
- Analytical modeling
- Pipeline automation
- User-driven analytics interface

## Goal
Showcase how raw domain-specific data can be transformed into actionable analytics using automated workflows.
